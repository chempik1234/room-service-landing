// Install SDK
npm install @roomservice/sdk

// Connect and create room
import { RoomService } from '@roomservice/sdk'

const client = new RoomService({
  apiKey: 'rs_live_your_key'
})

// Create real-time room
const room = await client.createRoom({
  maxUsers: 10
})

// Join and start streaming
await room.join({ id: 'user123' })
room.on('userJoined', (user) => {
  console.log(`${user.name} joined!`)
})
Features

Everything You Need for Real-Time Apps

Built by frontend developers, for frontend developers

Blazing Fast Setup

Get started in under 2 minutes. No backend configuration needed. Just install the SDK and start building.

Room-Based Architecture

Designed for rooms: game lobbies, meeting rooms, collaborative spaces. Not generic document stores.

Real-Time Sync

Bi-directional gRPC streams for instant updates. No polling, no Webhooks complexity.

Type-Safe SDK

Full TypeScript support with auto-completion. Catch errors at compile time, not runtime.

Flexible Data Model

Store any JSON data: lists, maps, nested objects. Perfect for game state, documents, or custom data.

User Presence

Built-in user management. Join/leave rooms, track presence, handle disconnections gracefully.
